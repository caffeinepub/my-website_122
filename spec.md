# Specification

## Summary
**Goal:** Build a responsive multi-section landing page with a bold dark charcoal and warm amber visual theme.

**Planned changes:**
- Sticky top navigation bar with links to Home, About, Features, and Contact sections
- Hero section with headline, subheading, call-to-action button, and background image
- About section describing the site's purpose
- Features/Services section with icon cards highlighting key points
- Contact/Footer section with placeholder contact info and copyright text
- Dark charcoal background with warm amber accent color palette throughout
- Clean sans-serif typography with consistent spacing and generous whitespace
- Hover state transitions on buttons, nav links, and cards
- Fully responsive layout for mobile and desktop

**User-visible outcome:** Users can view a polished, professional single-page website and navigate between all sections via the sticky nav bar.
